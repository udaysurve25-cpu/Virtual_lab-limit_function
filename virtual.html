Theoretical Foundations of Mathematical Functions

Classification of Mathematical Functions

  • Algebraic Functions: Polynomials, rational functions, radical functions
  • Transcendental Functions: Trigonometric, exponential, logarithmic
  • Hyperbolic Functions: sinh, cosh, tanh and their inverses
  • Special Functions: Gamma, Beta, Bessel, Error functions
  • Piecewise Functions: Functions defined by different formulas on different intervals
  • Periodic Functions: Functions that repeat values at regular intervals

Limit Properties for Different Function Types

For Continuous Functions: limx→a f(x) = f(a)

For Discontinuous Functions: One-sided limits may differ

For Functions with Asymptotes: Limits approach ±∞

Important Limit Theorems

  • Squeeze Theorem: Useful for trigonometric limits
  • L'Hôpital's Rule: For 0/0 or ∞/∞ indeterminate forms
  • Continuity Theorems: Sum, product, quotient of continuous functions
  • Intermediate Value Theorem: For continuous functions on closed intervals

Special Limits

limx→0 sin(x)/x = 1
limx→0 (1 - cos(x))/x = 0
limx→∞ (1 + 1/x)^x = e
limx→0 (e^x - 1)/x = 1
